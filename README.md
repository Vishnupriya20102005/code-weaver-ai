# code-weaver-ai
AI Web App Generator using OpenAI
🚀 Code Weaver - AI Web App Generator

🔗 Live Demo

https://code-weaver--srijasece2023.replit.app

📌 Description

Code Weaver is an AI-powered system that generates complete full-stack web applications from natural language input.

⚙️ Features

- 🧠 AI-based code generation
- 🌐 Full-stack web app creation
- ⚡ Fast API response system
- 🔄 REST API integration

🛠 Tech Stack

- Node.js
- Express.js
- OpenAI API

🚀 How to Run

1. Clone the repository
2. Install dependencies: "npm install"
3. Add your OpenAI API key in ".env"
4. Run the server: "npm start"

📁 Project Structure

- "server.js" → Backend API
- "frontend.js" → Client-side request handling
- "package.json" → Dependencies
